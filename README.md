Searchkick Boost Failing
========================

This repository is an example Rails app running with the following setup:

* Ruby 2.1.3
* Rails 4.1.6
* Searchkick 0.8.3

## Installation

* Clone this repository
* `bundle install`
* `rails s`
* Visit localhost:3000 and see the elasticsearch error
